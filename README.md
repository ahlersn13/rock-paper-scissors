# Rock Paper Scissors

A two-player Rock Paper Scissors game built with vanilla JavaScript. Each round randomly generates choices for both players, displays the corresponding icons, and determines the winner.

## Features

- Randomized player selections each round with animated icon updates
- Dynamic win/loss/draw result display
- Responsive design that adapts to mobile and desktop screens
- Custom styled UI with color-coded player labels

## Tech Stack

- **Languages:** JavaScript, HTML, CSS

## Getting Started

No build step required — just clone and open in your browser.

```bash
git clone https://github.com/ahlersn13/rock-paper-scissors.git
cd rock-paper-scissors
open index.html
```
